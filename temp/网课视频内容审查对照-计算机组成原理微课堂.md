# 计算机组成原理微课堂：视频内容审查与本校 PPT 对照

## 审查方式

我没有只看标题。这次对 `/Users/ming/Downloads/downloadedVideo/计算机组成原理微课堂` 下全部 154 个 mp4 做了内容抽样：

- 每期视频抽取早段、中段、后段 3 张画面。
- 合成 7 张 contact sheet：p001-p020、p021-p040、p041-p060、p061-p083、p084-p105、p106-p132、p133-p154。
- 逐组目检课件画面和板书内容，再和本地学校 PPT、Homework1-5、Lab1-4 对照。

视频没有字幕轨，所以这次判断基于视频画面中的课件内容，不是完整音频转写。

## 总体结论

这个网课符合“计算机组成原理”的大范围，但不是完全按你们学校这门课的重点组织。它更像中文本科“计组全套课”，覆盖：

- 计算机系统概论。
- 数据表示与运算。
- 存储器和 Cache。
- 指令系统。
- CPU、控制器、流水线。
- 总线。
- I/O。
- 中断。

而你们学校本学期 PPT/Homework/Lab 的考试主线更窄，主要是：

- PPT 01：计算机抽象、性能公式。
- PPT 02：ISA、MIPS、R/I/J 指令格式、机器码、寻址、branch/jump。
- PPT 03：补码、IEEE 754、运算。
- PPT Chapter04：MIPS 数据通路、控制、single-cycle、multi-cycle、pipeline。
- PPT 05：存储层次、Cache、地址映射。

所以这个网课能补中文理解，但不能直接替代学校 PPT。尤其是 MIPS 指令编码和学校的 MIPS datapath，还是必须以学校 PPT、Homework2、Lab3、Homework4 为准。

## 不建议考前主看

这些我看过画面，确认内容确实比你们学校当前 PPT/Homework 主线更深或更偏。

### 算术硬件过深

- p22-p24 移位运算细节。
- p27-p30 全加器、串行进位、先行进位加法器。
- p31-p38 定点乘法、除法硬件实现。
- p41-p42 浮点乘法、除法。

你们学校 PPT 03 和 Homework3 重点更像“表示和计算”，不是设计乘除法器。

### 存储器芯片与扩展过深

- p45、p47-p48 存储器分类、主存结构、译码结构。
- p51-p52 SRAM 阵列和芯片实例。
- p55-p64 ROM、位扩展、字扩展、字位扩展、双端口、多体交叉。

这些在中文计组课里常见，但你们本学期 Homework5 明显更偏 Cache 地址映射。

### 虚拟存储器不是主线

- p77-p83 虚拟存储器、页表、TLB。

你们学校 PPT 05 的 memory hierarchy 图里可能出现 TLB，但 Homework/Lab 没把虚拟存储器作为核心计算题。

### 控制器/微程序/中断不是主线

- p107-p118 硬布线控制器、微程序控制器。
- p119-p124 异常与中断。

你们 PPT Chapter04 关注 MIPS datapath、control、pipeline，不建议考前花大量时间看微程序控制器。

### 总线和 I/O 基本不看

- p133-p142 总线。
- p143-p154 I/O 系统、I/O 接口、查询、中断、DMA。

这些画面确认都是总线/I/O 章节。你们本地学校 PPT 和 Homework1-5 没把它们作为主范围，所以考前直接跳过。
