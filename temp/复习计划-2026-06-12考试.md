## 优先级

### P0 必会

这些内容最像作业题和实验题，必须会算、会写步骤。

1. Chapter 1：计算机抽象、性能公式
   - 七个计算机体系结构思想。
   - 高级语言程序到机器执行的转换流程。
   - Frame buffer 大小计算。
   - CPU execution time = Instruction Count × CPI × Clock Cycle Time。
   - Performance = 1 / Execution Time。
   - 比较两个编译器、两个 CPU 的 CPI、clock rate、execution time。

2. Chapter 2：ISA 与 MIPS
   - ISA、architecture、organization 的区别。
   - RISC/CISC 基本区别。
   - MIPS 32 个寄存器，尤其 `$zero`、`$t0-$t9`、`$s0-$s7`、`$a0-$a3`、`$v0-$v1`、`$sp`、`$ra`。
   - `add/sub/and/or/xor/nor/slt/sll/addi/lw/sw/beq/bne/j/lui/ori/jr`。
   - R/I/J 三种指令格式。
   - op、rs、rt、rd、shamt、funct 的位数和含义。
   - 二进制机器码、十六进制机器码、汇编指令互转。
   - big-endian/little-endian。
   - byte address、word alignment、offset 地址计算。
   - branch offset、PC + 4、jump address 的计算。

3. Chapter 3：数据表示与运算
   - 二进制、十进制、十六进制互转。
   - unsigned、sign-magnitude、one's complement、two's complement。
   - 补码加减法、溢出判断。
   - sign extension。
   - IEEE 754 single precision：sign、exponent、fraction、bias=127。
   - 十进制小数和 IEEE 754 表示互转，例如 `-0.75`、`0x0C000000` 这种题。

4. Chapter 4：处理器
   - MIPS register file、instruction memory、data memory、ALU、PC、control。
   - single-cycle datapath 的取指、译码、执行、访存、写回路径。
   - `lw/sw/R-type/beq` 在数据通路中的路径。
   - 控制信号：RegWrite、MemRead、MemWrite、MemtoReg、ALUSrc、RegDst、Branch、ALUOp。
   - Homework4 的 LWI：`Reg[Rt] = Mem[Reg[Rd] + Reg[Rs]]`，会分析可复用模块和新增模块。
   - single-cycle、multi-cycle、pipeline 的优缺点。
   - 五级流水线 IF、ID、EX、MEM、WB。
   - pipeline registers：IF/ID、ID/EX、EX/MEM、MEM/WB。
   - structural/data/control hazards、stall、flush、forwarding。

5. Chapter 5：存储层次与 Cache
   - memory hierarchy、locality：temporal locality、spatial locality。
   - SRAM、DRAM、disk 的速度/容量/价格关系。
   - cache block、tag、index、offset、valid bit。
   - direct mapped、fully associative、set associative。
   - 地址格式计算：主存大小、cache 大小、block size、associativity 给定时，求 tag/index/offset。
   - hit、miss、miss rate。
   - write-through、write-back、write allocate。
   - replacement：random、LRU。
   - Cache 容量、相联度、块大小对 miss rate 的影响。
