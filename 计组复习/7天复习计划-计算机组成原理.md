如果每天时间少于 4 小时，优先保：MIPS、IEEE 754、数据通路、流水线、Cache。

## Day 7：Cache + 总复盘

- `学校文件/PPT/Computer Organization 05 for 24H.pdf`
- `学校文件/PPT/Computer Organization 05 Part1 for 24H.pdf`

- `学校文件/Lab4 for 24H/Lab4 Cache Performance for 24H.pdf`

网课：

- p65-p76：Cache 全部重点

任务：

- 看学校 PPT 05 的 memory hierarchy、locality、Cache。
- 看 p65-p76，重点是：
  - direct mapping
  - fully associative
  - set associative
  - tag/index/offset
  - replacement
  - write policy
- 重做 Homework5。
- 练 4 道 Cache 地址划分题：
  - direct mapped
  - fully associative
  - 2-way set associative
  - 不同 block size 情况。
- 看 Lab4，记住：
  - Cache capacity 变大，一般 miss rate 降低。
  - associativity 增大，一般 conflict miss 降低。
  - block size 不是越大越好，过大可能增加 miss penalty 或浪费。

当天必须产出：

- 一张 Cache 地址划分模板。
