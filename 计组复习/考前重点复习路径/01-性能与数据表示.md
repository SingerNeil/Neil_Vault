## 做题重点

### 性能题

必须会写：

```text
CPU Time = IC * CPI * Clock Cycle Time
CPU Time = IC * CPI / Clock Rate
CPI = Execution Time / (IC * Clock Cycle Time)
Average CPI = sum(IC_i * CPI_i) / sum(IC_i)
```

看到比较两个 compiler / processor：

```text
ClockRate_A / ClockRate_B = (IC_A * CPI_A) / (IC_B * CPI_B)
```

Frame buffer：

```text
bytes = width * height * bytes per pixel
RGB 每色 8 bits -> 每像素 3 bytes
```

### 原码 / 反码 / 补码 / 移码

考试只按题型准备：

1. 最高位判断符号。
2. 正数编码基本一样。
3. 负数补码：原数绝对值写二进制，按位取反，加 1。
4. 补码转十进制：最高位 1 时，用 `unsigned value - 2^n`。
5. 移码常用于阶码，bias 形式：真实值 + 偏置。

### IEEE754

Single precision 必背：

```text
sign = 1 bit
exponent = 8 bits
fraction = 23 bits
bias = 127
value = (-1)^S * 1.F * 2^(E - 127)
```

Hex 转 IEEE754：

1. 写成 32-bit binary。
2. 切 S / E / F。
3. 算 `E - 127`。
4. 写最终值。
