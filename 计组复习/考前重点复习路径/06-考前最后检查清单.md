## 1. 必背公式

```text
CPU Time = IC * CPI * Clock Cycle Time
CPU Time = IC * CPI / Clock Rate
CPI = Execution Time / (IC * Clock Cycle Time)
Average CPI = sum(IC_i * CPI_i) / sum(IC_i)
Speedup = Old Time / New Time
Frame Buffer = width * height * bytes per pixel
```

## 2. IEEE754

```text
single precision:
sign = 1 bit
exponent = 8 bits
fraction = 23 bits
bias = 127
value = (-1)^S * 1.F * 2^(E - 127)
```

做题顺序：

```text
hex -> 32-bit binary -> S/E/F -> E-127 -> value
```

## 3. Cache

```text
address bits = log2(main memory size)
offset = log2(block size)
cache blocks = cache size / block size
direct index = log2(cache blocks)
set index = log2(cache blocks / ways)
tag = address bits - index - offset
```

格式：

```text
Direct: Tag | Index | Offset
Fully associative: Tag | Offset
Set associative: Tag | Set index | Offset
```

## 4. Pipeline

```text
IF -> ID -> EX -> MEM -> WB
```

四个 pipeline registers：

```text
IF/ID
ID/EX
EX/MEM
MEM/WB
```

Hazard：

```text
Structural: hardware resource conflict
Data: use result before ready
Control: branch/jump changes PC
```

解决：

```text
stall
forwarding
flush
branch prediction
```

## 5. ALU / 运算器

```text
A = 55H
W = 33H
A + W = 88H
A - W = 22H
A OR W = 77H
A AND W = 11H
~A = AAH
```

补码溢出：

```text
正 + 正 = 负 -> overflow
负 + 负 = 正 -> overflow
```

## 6. MIPS 保底

```text
R-format: op rs rt rd shamt funct
I-format: op rs rt immediate
J-format: op address
```

```text
Branch Target = PC + 4 + offset * 4
offset = (Target - (PC + 4)) / 4
Jump Target = {PC+4 高4位, address field, 00}
```

```text
Big-endian: 最高有效字节放低地址
Little-endian: 最低有效字节放低地址
```
