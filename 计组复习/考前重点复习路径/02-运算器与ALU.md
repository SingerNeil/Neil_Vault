## 做题重点

### 补码加减法

减法统一变加法：

```text
A - B = A + (-B)
```

`-B` 用补码表示。

溢出判断：

```text
同号相加，结果异号 -> overflow
异号相加通常不溢出
```

也可以记：

```text
正 + 正 = 负 -> overflow
负 + 负 = 正 -> overflow
```

### 移位题

逻辑移位：

```text
空位补 0
```

算术右移：

```text
符号位保持不变
```

循环移位：

```text
移出去的位绕回另一端
```

### ALU 实验表

已知：

```text
A = 55H
W = 33H
```

必须会：

| S2 S1 S0 | 功能 | 结果 |
|---|---|---|
| 000 | A + W | 88H |
| 001 | A - W | 22H |
| 010 | A OR W | 77H |
| 011 | A AND W | 11H |
| 100, C=0 | A + W + C | 88H |
| 100, C=1 | A + W + C | 89H |
| 101, C=0 | A - W - C | 22H |
| 101, C=1 | A - W - C | 21H |
| 110 | ~A | AAH |
| 111 | A | 55H |

解释句：

```text
Register A/W 写入依赖 CK rising edge；ALU 是组合逻辑，A、W 和控制信号稳定后立即输出。
```
