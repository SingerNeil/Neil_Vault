### 只背这些

```text
R-format: op rs rt rd shamt funct
I-format: op rs rt immediate
J-format: op address
```

Branch：

```text
Branch Target = PC + 4 + offset * 4
offset = (Target - (PC + 4)) / 4
```

Jump：

```text
Jump Target = {PC+4 高4位, address field, 00}
```

Endian：

```text
Big-endian: 最高有效字节放低地址
Little-endian: 最低有效字节放低地址
```

### LWI 答题模板

题目：

```text
LWI Rt, Rd(Rs)
Reg[Rt] = Mem[Reg[Rd] + Reg[Rs]]
```

复用模块：

```text
Instruction Memory
Register File
ALU
Data Memory
MemtoReg write-back path
PC update logic
main control
```

新增需求：

```text
需要让 Register File 第二个 read address 能选择 rd field。
在第二个 read-register address 前加一个 mux。
控制单元增加 LWI 译码规则。
```

控制信号：

```text
RegWrite = 1
MemRead = 1
MemWrite = 0
MemtoReg = 1
ALUSrc = 0
ALUOp = add
Write register = Rt
Second source register = Rd
```

## 3. 总线 / I/O / 微程序控制器

```text

```
