# 集市记账 APP 待修清单

## P1 关联活动选择器不适合活动变多后的场景
- 模块：timeline
- 当前现象：
  当前“关联活动（可选）”已经不再手填 event_id，但仍然以平铺选项方式展示活动。
- 为什么是问题：
  当活动数量增加后，用户会很难快速找到目标活动，页面也会变得拥挤。
- 是否阻塞主线：否
- 建议处理时机：
  在首页真实聚合完成后，作为 timeline 表单体验优化处理。
- 后续给 Codex 的工单方向：
  将“关联活动”改为可点击选择器，支持“不关联活动”、活动名称 + 日期范围展示。

## P2 timeline 页面文件体积过大，后续需要拆分
- 模块：timeline
- 当前现象：
  `app/timeline/[id].tsx` 和 `app/timeline/new.tsx` 代码量较大。
- 为什么是问题：
  后续继续加功能会降低可维护性，影响局部修改效率。
- 是否阻塞主线：否
- 建议处理时机：
  在主链路闭环完成后统一做一次重构。
- 后续给 Codex 的工单方向：
  抽离表单区块和详情展示区块，减少页面文件体积。

## P1 首页目前仍缺真实聚合展示
- 模块：home
- 当前现象：
  首页仍不是完整真实数据首页。
- 为什么是问题：
  用户虽然可以录数据，但还看不到本月收入、支出、现金结余、最近活动复盘的价值。
- 是否阻塞主线：是
- 建议处理时机：
  下一轮立即处理。
- 后续给 Codex 的工单方向：
  实现首页真实聚合：本月收入、本月支出、现金结余、最近活动复盘。
